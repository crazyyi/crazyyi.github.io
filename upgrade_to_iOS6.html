Upgrading ChinesePod to iOS 6

Wed 29 January 2014
By Yi Zeng

Upgrading from iOS 5 - iOS 6.

This seems to be easy on ChinesePod. All I did was to change the compile target from 5.0 to 6.0.

Then if it says dispatch_release error just use the following:

/*header*/
  #if __IPHONE_OS_VERSION_MIN_REQUIRED >= 60000
      #define NEEDS_DISPATCH_RETAIN_RELEASE 0
  #else
      #define NEEDS_DISPATCH_RETAIN_RELEASE 1
  #endif

/* implementation */
  #if NEEDS_DISPATCH_RETAIN_RELEASE
      dispatch_release(self.reachabilitySerialQueue);
  #endif
